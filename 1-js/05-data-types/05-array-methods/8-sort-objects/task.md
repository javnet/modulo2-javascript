# Sort objects

Escriba la funci√≥n `sortByName(users)` que obtiene una matriz de objetos con la propiedad `name` y la ordena.

Por ejemplo:
```js no-beautify
let john = { name: "John", age: 25 };
let pete = { name: "Pete", age: 30 };
let mary = { name: "Mary", age: 28 };

let arr = [ john, pete, mary ];

sortByName(arr);

// now: [john, mary, pete]
alert(arr[1].name); // Mary
```
