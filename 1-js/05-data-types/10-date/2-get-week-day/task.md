# Show a weekday

Escribe una función `getWeekDay(date)` para mostrar el día de la semana en formato corto.: 'MO', 'TU', 'WE', 'TH', 'FR', 'SA', 'SU'.

Por ejemplo:

```js no-beautify
let date = new Date(2012, 0, 3); // 3 Jan 2012
alert(getWeekDay(date)); // should output "TU"
```
